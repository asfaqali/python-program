# If Else Program

## Description
This program shows how to use the **if–else statement**.
It helps the program make a decision based on a condition.

If the condition is true, one action is done.  
If the condition is false, another action is done.

## Purpose
- Learn basic programming logic
- Understand decision making using if–else
- Good for beginners

## How It Works
1. The user enters a value.
2. The program checks the condition using `if`.
3. If the condition is true, the `if` block runs.
4. If the condition is false, the `else` block runs.
